# 02_主流程速记

快速阅读摘要。正式定义以 [02_引入Agent网关版方案.md](../02_引入Agent网关版方案.md)、[04_接口设计.md](../04_接口设计.md) 和 [05_策略中心设计.md](../05_策略中心设计.md) 为准。

## 1. base 登录

```mermaid
sequenceDiagram
    participant 用户 as 用户浏览器
    participant Agent as 业务 Agent
    participant GW as Agent 网关
    participant IDaaS as IDaaS

    用户->>Agent: 打开页面
    Agent-->>用户: 无 site_session，跳转 /gw/auth/login
    用户->>GW: 发起 base 登录
    GW-->>用户: 302 到 IDaaS /authorize?scope=base
    用户->>IDaaS: GET /oauth2/authorize
    IDaaS-->>用户: 展示登录页或快速回调
    用户->>IDaaS: 提交登录信息（如需要）
    IDaaS-->>用户: 302 回网关 callback
    用户->>GW: GET /gw/auth/base/callback?code&state
    GW->>GW: 生成 ticketST，绑定 code
    GW-->>用户: 回跳业务 Agent，只带 ticketST + state
    用户->>Agent: 回原页面
    Agent->>GW: 后端用 ticketST 换用户信息
    GW->>IDaaS: code 换 Tc 短 access_token
    GW->>IDaaS: 用 Tc 调用户信息接口
    GW-->>Agent: 返回用户信息
    Agent->>Agent: 创建 site_session
```

## 2. 资源请求 / 获取 TR

```mermaid
sequenceDiagram
    participant 用户 as 用户浏览器
    participant Agent as 业务 Agent
    participant GW as Agent 网关
    participant PC as 策略中心
    participant IDaaS as IDaaS
    participant IAM as IAM
    participant MCP as MCP 网关

    用户->>Agent: 发起业务请求
    Agent->>Agent: 检查本地 TR

    alt 本地 TR 可复用
        Agent->>MCP: 带 TR 调 MCP
    else 本地无可用 TR
        Agent->>GW: POST /gw/token/resource-token(required_tools)
        GW->>PC: resolve-by-tools
        PC-->>GW: requiredPermissionPointCodes
        GW-->>Agent: 返回 redirect_url + request_id
        Agent-->>用户: 跳转授权
        用户->>GW: GET /gw/auth/authorize
        GW-->>用户: 302 到 IDaaS /authorize
        用户->>IDaaS: GET /oauth2/authorize
        IDaaS-->>用户: 首次授权展示授权页
        用户->>IDaaS: 勾选并确认授权
        IDaaS-->>用户: 302 回网关 callback
        用户->>GW: GET /gw/auth/consent/callback?code&state
        GW->>IDaaS: code 换 Tc
        GW->>IAM: 申请 T1
        GW->>IAM: 申请 TR
        GW-->>用户: 回跳业务 Agent，只带 token_result_ticket
        用户->>Agent: 回原页面
        Agent->>GW: 后端用 token_result_ticket 换 TR
        Agent->>MCP: 带 TR 调 MCP
    end

    MCP->>PC: resolve-by-tools(currentTool)
    MCP->>PC: query strategies(requiredPermissionPointCodes)
    MCP->>PC: resolve-by-codes(TR.agency_user.consented_scopes[].code)
```

## 3. MCP 网关运行时判定

当前工具可调用，必须同时满足：

1. 当前工具所需权限点已经在 `TR.agency_user.consented_scopes` 中。
2. 当前用户通过这些权限点对应的 Agent 策略判断。
3. 当前请求工具包含在 `TR` 权限点反查出的工具集合中。

如果任一条件不满足，当前请求直接失败。
