这份拆分文档采用统一设计口径：

- 只覆盖默认方案
- 浏览器入口是 `ALB -> 接入Agent`
- `Agent网关` 是内部安全编排服务
- 接入 Agent 前端只负责页面跳转和回到原页面
- 接入 Agent 后端/BFF 负责：
  - 调用 `Agent网关`
  - 调用 `MCP`
  - 暂存 `request_id`
  - 暂存 `security_session_id`
  - 暂存当前 `TR`
- `Tc/T1` 只保留在 `Agent网关`
- `TR` 过期后由接入 Agent 后端/BFF 回 `Agent网关` 刷新
- 对接入 Agent 的运行时外观只保留 3 个接口：
  - `POST /api/v1/runtime/tr/acquire`
  - `POST /api/v1/runtime/security-sessions/{security_session_id}/tr/refresh`
  - `POST /api/v1/mcp/invoke`
