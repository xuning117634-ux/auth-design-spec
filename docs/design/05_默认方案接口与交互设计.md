这份拆分文档采用统一设计口径：

- 只覆盖默认方案
- 浏览器入口是 `ALB -> 接入Agent`
- `Agent网关` 是内部安全编排服务
- `Agent` 只透传并暂存 `session_id + 当前 TR`
- `Tc/T1` 只保留在 `Agent网关`
- `TR` 过期后由 `Agent` 回 `Agent网关` 刷新
- 对接入 Agent 的运行时外观只保留 3 个接口：
  - `POST /api/v1/runtime/tr/acquire`
  - `POST /api/v1/runtime/sessions/{session_id}/tr/refresh`
  - `POST /api/v1/mcp/invoke`
- 登录、授权、令牌获取和资源访问都按模块间正式接口来描述
